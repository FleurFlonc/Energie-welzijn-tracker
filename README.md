# Energie & Welzijn (ADD) Tracker — PWA + GitHub Pages

Deze tracker is bedoeld om je **energie, welzijn, cyclus, voeding en ADD-klachten** bij te houden (privacy-first: alles lokaal).

## GitHub Pages
1. Nieuwe repo → upload alle bestanden naar de root.
2. Settings → Pages → Deploy from branch → `main` / `/ (root)`.
3. Open op iPhone (Safari) → Deel → **Zet op beginscherm** (PWA/offline).

## Backup
Gebruik **Export/Import** om data te kopiëren naar Notes/Drive en later terug te zetten.
